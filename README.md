# ❤️ Предсказание сердечно-сосудистых заболеваний

Веб-приложение для оценки риска развития сердечно-сосудистых заболеваний на основе клинических данных пациента. Приложение использует ансамбль моделей машинного обучения для предсказания и предоставляет персональные рекомендации.

## 📊 Результаты модели

| Метрика | Значение |
|---------|----------|
| **Точность (Accuracy)** | **92.39%** |
| **Чувствительность (Sensitivity)** | 94.12% |
| **Специфичность (Specificity)** | 90.24% |
| **F1-Score** | 93.20% |
| **Precision** | 92.31% |

## 📋 Используемые признаки

| Признак | Описание | Возможные значения |
|---------|----------|-------------------|
| **Age** | Возраст | 0-120 лет |
| **Sex** | Пол | M (мужской), F (женский) |
| **ChestPainType** | Тип боли в груди | ATA, NAP, ASY, TA |
| **RestingBP** | Систолическое давление | мм рт. ст. |
| **Cholesterol** | Уровень холестерина | мг/дл |
| **FastingBS** | Глюкоза натощак >120 | 0 (нет), 1 (да) |
| **RestingECG** | Результаты ЭКГ | Normal, ST, LVH |
| **MaxHR** | Максимальная ЧСС | 0-250 |
| **ExerciseAngina** | Стенокардия при нагрузке | N (нет), Y (да) |
| **Oldpeak** | Депрессия ST | 0.0-10.0 |
| **ST_Slope** | Наклон ST сегмента | Up, Flat, Down |
